ประกาศความระมัดระวังและช่องทางตรวจสอบเว็บ "klogic-legacy.vercel.app"

ช่องทางตรวจสอบเพื่อความโปร่งใส: GitHub

เว็บ klogic-legacy.vercel.app เป็นส่วนของโครงการที่นำเว็บไซต์เก่ากลับมา (klogic2.kmutnb.ac.th ที่ได้ถูกย้ายไป reg.kmutnb.ac.th) ซึ่งเว็บไซต์นี้เป็นเครื่องมือเพื่อให้ผู้ใช้เข้าถึงและจัดการข้อมูลส่วนตัวของนักศึกษา เราไม่สามารถรับผิดชอบในกรณีที่ท่านเปลี่ยนแปลงข้อมูลที่ถูกกรอกเข้าไปในเว็บไซต์นี้แล้ว เกิดผลกระทบต่อตัวท่านเอง ดังนั้นควรหลีกเลี่ยงการแก้ไขข้อมูลใด ๆ บนเว็บไซต์นี้

โครงการนี้ได้รับการพัฒนาเพื่อเป็นแนวทางในการศึกษาการทำ HTTP Request เท่านั้น และไม่มีเจตนาในการคัดลอกหรือดัดแปลงผลงาน โปรดทราบว่าเว็บไซต์นี้อาจมีความผิดพลาด (เช่น การป้อนรหัสผ่านไม่ถูกต้องหรือปัญหาการออกจากระบบที่อาจนำไปสู่เว็บไซต์ใหม่) และเราไม่เก็บข้อมูลใด ๆ บนเว็บไซต์นี้
